# Stock Analyzer

`Stock Analyzer` 是一个股票价值分析 skill，用于对单只股票做结构化价值判断，并输出可直接交付的分析报告。

它主要做这些事：

- 评估行业、公司、估值和逆向机会
- 识别护城河、定价权和估值陷阱
- 按不同市场和行业规则调整分析口径
- 支持多方法论并行输出
- 给出买入、卖出、止损、止盈和持有期建议

当前支持的方法论：

- 《投资中最简单的事》
- `Mr Dang投资理念方法`

## 触发方式

可直接用这些提示词触发：

```text
分析 SZ000960
```

```text
用 Stock Analyzer 分析 000960 并生成报告
```

```text
分析这只股票，重点看行业、公司、估值和买卖价格
```

```text
用 Mr Dang投资理念方法 分析 SZ002170
```

## 输出

该 skill 会生成结构化报告，通常包含：

- 方法论结论总览
- 股票信息与关键数据
- 近 30 天重大事件
- 行业分析
- 公司分析
- 估值分析
- 逆向投资与定价权
- 分方法论投资结论
- 风险提示
- 交卷前自检摘要

## 核心约束

- 不把目标价、52 周区间、分析师预期误当现价
- 不引用单日资金流向
- 不忽略非标签业务
- 不混用不同市场和行业的分析口径
- 报告完成后必须自检
