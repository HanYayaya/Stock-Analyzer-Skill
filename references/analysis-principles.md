# 分析原则补充约束

本文件只保留跨行业通用的补充约束。行业属性相关的细则已经拆分为独立 reference：

- `references/cyclical-industry-rules.md`
- `references/non-cyclical-industry-rules.md`

## 1. 多业务板块必须全面分析

分析公司时，不能只围绕最知名或标签化业务展开，必须逐一覆盖所有主要业务板块。

最低要求：

- 列出主要业务板块的营收、占比、毛利率、增速
- 分别评估各板块的竞争地位、护城河和增长前景
- 关注当前占比不高但有战略价值的新兴业务
- 分析各板块之间的协同效应和一体化优势

禁止行为：

- 因为某个板块当前利润低或占比小就直接忽略
- 只分析市场最熟悉的单一板块，再用它替代全公司判断

## 2. 分析必须纳入宏观与地缘因素

公司不在真空中经营。分析前必须补看与公司或行业直接相关的宏观和时事因素。

重点包括：

- 大宗商品价格
- 汇率
- 利率与信用环境
- 贸易政策
- 地缘冲突
- 关键运输通道和供应链风险

最低要求：

- 在行业分析和公司分析中说明这些因素对各业务板块的影响
- 在估值分析中评估其对短中期利润的影响
- 在风险提示中明确列出当前时事相关的具体风险

## 3. 战略投资 / 隐蔽资产只能锦上添花

公司持有热门资产或未上市股权时，只能作为补充加分项，不能成为核心买入逻辑。

最低要求：

- 先判断主业是否通过“三好”基本门槛
- 未上市公司估值默认打 5-7 折
- 早期项目估值默认打约 3 折
- 战略投资带来的估值提升不应超过总分的 5-10%

核心检验：

```text
去掉这部分战略投资价值后，公司主业是否仍值得买入？
```

如果答案是否定，则该逻辑更接近概念炒作，而不是价值投资。

## 4. 禁止引用单日资金流向

以下内容不属于价值投资分析所需信息，不得纳入报告：

- 今日主力资金净流入 / 净流出
- 特大单、大单、小单、散单净流入
- 盘中资金博弈类短线信号

允许但需区分的内容：

- 中长期股东结构变化
- 大股东增持减持
- 新进或退出的重要机构股东

不要把中长期股东结构信号混写成“资金流向”。
