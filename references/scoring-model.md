# 评分细则

本文件定义 `Stock Analyzer` 的评分框架、权重、加减分逻辑和一票否决规则。

## 1. 总体结构

综合得分由三大基础模块和两个调整项组成：

```text
综合得分 = 行业分 × 30% + 公司分 × 35% + 估值分 × 25% + 逆向调整 + 定价权调整
```

默认目标是中长期价值投资判断，而不是短期交易判断。

在进入正式评分前，先做两层分类：

1. 先判市场：A 股 / 港股 / 美股或 ADR
2. 再判行业：周期性 / 非周期性

## 2. 行业分析（权重 30%）

行业分析总分 100，必须拆成四项：

| 维度 | 分值 | 判断重点 |
|---|---:|---|
| 行业格局 | 25 | 是寡头格局还是高度内卷 |
| 定价权环境 | 25 | 行业整体能否顺价或主动提价 |
| 需求稳定性 | 25 | 需求是刚需、周期、还是易被替代 |
| 进入壁垒 | 25 | 资本、牌照、技术、渠道、品牌壁垒 |

粗略解释：

- 80-100：好行业
- 60-79：可接受行业
- 40-59：一般行业
- 0-39：差行业

## 3. 公司分析（权重 35%）

公司分析总分 100，必须拆成四项：

### 3.1 护城河（40 分）

| 子项 | 分值 |
|---|---:|
| 品牌护城河 | 10 |
| 成本护城河 | 8 |
| 转换成本 | 8 |
| 网络效应 | 8 |
| 牌照 / 专利 / 监管壁垒 | 6 |

### 3.2 盈利能力（20 分）

重点看：

- 毛利率
- 净利率
- ROE
- 与行业平均对比

### 3.3 财务健康（20 分）

重点看：

- 资产负债率
- 有息负债压力
- 经营现金流
- 自由现金流

### 3.4 管理层质量（20 分）

重点看：

- 诚信
- 执行能力
- 股东回报意识
- 战略清晰度

补充约束：

- 必须覆盖所有主要业务板块，不能只分析标签业务
- 要评估各板块的营收、占比、毛利率、增速、竞争地位和战略价值
- 要分析板块之间的协同效应
- 行业属性判断前，先区分周期性与非周期性，并读取对应行业规则文件
- 市场属性判断前，先区分 A 股、港股、美股 / ADR，并读取对应市场规则文件

## 4. 估值分析（权重 25%）

估值分析总分 100，必须拆成四项：

| 维度 | 分值 | 判断重点 |
|---|---:|---|
| PE 所处水平及原因解释 | 25 | 便宜是否有原因，贵是否有支撑 |
| 估值陷阱检查 | 25 | 低 PE 是否掩盖基本面恶化 |
| 辅助指标 | 25 | PB、PS、PEG、EV/EBITDA |
| 历史分位与安全边际 | 25 | 当前估值处于历史什么位置 |

估值结论不能只看“低不低”，必须回答：

- 为什么低
- 低估是否可逆
- 是否可能是价值陷阱

补充约束：

- 对 A 股，按 `a-share-market-rules.md` 处理政策、治理和本土估值习惯
- 对港股，按 `hk-market-rules.md` 处理流动性折价、AH 口径和高股息判断
- 对美股 / ADR，按 `us-market-rules.md` 处理 GAAP、FCF、回购和全球化估值逻辑
- 对周期行业，按 `cyclical-industry-rules.md` 处理周期阶段和估值口径
- 对非周期行业，按 `non-cyclical-industry-rules.md` 处理质量与持续性判断
- 战略投资 / 隐蔽资产只能作为补充加分，不得替代主业估值判断

## 5. 逆向投资调整（建议范围 -10 到 +10）

逆向投资不是固定加分项，只在市场情绪极端时使用。

三重验证：

1. 行业还好吗
2. 公司还好吗
3. 价格真的足够便宜吗

建议调整：

| 情况 | 调整 |
|---|---:|
| 三项都通过，且市场明显悲观错杀 | +10 |
| 部分通过，需要继续观察 | 0 |
| 情绪悲观但基本面恶化 | -5 |
| 明显是价值陷阱 | -10 |

## 6. 定价权调整（建议范围 -10 到 +10）

定价权是重要但容易被低估的专项。

判断问题：

- 公司能否主动提价
- 提价后客户是否仍接受
- 提价是否能转化成利润，而不是被成本吞掉

建议调整：

| 定价权水平 | 表现 | 调整 |
|---|---|---:|
| 强 | 可主动提价且需求稳定 | +10 |
| 中 | 可跟随通胀提价 | +5 |
| 弱 | 提价明显伤害需求 | 0 |
| 无 | 价格完全被外部决定 | -10 |

## 6.1 战略投资 / 隐蔽资产加分上限

若公司持有热门资产、未上市股权或其他隐蔽资产，其加分必须受限：

- 仅在主业已通过“三好”基本门槛后才考虑加分
- 未上市公司估值默认打 5-7 折
- 早期项目估值默认打约 3 折
- 对总分的提升一般不超过 5-10 分中的小部分，不能单独扭转差主业结论

核心检验：

```text
去掉这部分战略投资价值后，公司主业是否仍值得买入？
```

## 7. 一票否决项

以下问题若成立，应优先否决，而不是继续靠总分粉饰：

| 检查项 | 默认门槛 |
|---|---|
| 行业得分过低 | < 30 |
| 护城河过弱 | < 10 |
| 存在严重财务造假或高度可信造假指控 | 直接否决 |
| 管理层诚信为负面 | 直接否决 |
| 经营现金流长期恶化且无合理解释 | 强烈否决 |
| 有息负债压力极高且再融资依赖重 | 强烈否决 |

如果触发一票否决，最终结论应优先写明否决原因，而不是只报综合分。

## 8. 输出解释建议

最终结论建议映射为：

| 分数区间 | 建议标签 |
|---|---|
| 85-100 | 强烈推荐 |
| 75-84 | 推荐 |
| 60-74 | 可关注 |
| 45-59 | 观望 |
| 30-44 | 不推荐 |
| 0-29 | 远离 |

如果触发一票否决，标签应向下覆盖。

## 9. 使用原则

- 评分是为了结构化判断，不是为了制造精确幻觉
- 分数必须能回溯到具体依据
- 若关键数据来源不足，应先降级置信度，再决定是否评分
- 对不同市场的专项修正，直接落到现有市场规则文件：
- `a-share-market-rules.md`
- `hk-market-rules.md`
- `us-market-rules.md`
- 不得把单日资金流向数据纳入加分、扣分、催化剂或风险依据
