# 美股市场规则

适用于美国上市公司及 ADR 标的。

## 1. 数据与口径

- 美股 / ADR 优先使用 `yfinance`
- 财务口径默认优先理解为 GAAP，再判断是否有调整后利润
- 对 ADR，必须区分 ADR 本身与底层本土业务，不得直接把 ADR 交易表现等同于经营表现

## 2. 分析重点

美股分析通常要更重视：

- 回购、股东回报与资本配置
- 管理层指引和市场预期差
- FCF 质量
- 竞争优势是否可全球化复制
- 反垄断、诉讼、会计调整等制度性风险

## 3. 估值习惯

- 美股对高质量、高壁垒公司给予更高估值溢价较常见
- 不能机械把高 PE 等同于高估，必须拆解成长持续性与资本回报
- 对 SaaS、平台、科技龙头，要补看 FCF、经营杠杆、回购与稀释

## 4. 风险提示

美股特有风险至少考虑：

- 利率与贴现率上升带来的估值压缩
- 指引不及预期带来的估值重定价
- 反垄断与诉讼
- ADR 结构与地缘政治风险
