# 港股市场规则

适用于港交所上市公司，包括纯港股、红筹、H 股及双重上市公司。

## 1. 数据与口径

- 港股优先使用 `yfinance`，必要时用 `AkShare` 补充
- 必须显式标注币种为 HKD
- 对 AH 双重上市公司，A/H 两端必须独立取数，禁止直接汇率换算推导估值

## 2. 分析重点

港股分析通常要更重视：

- 流动性折价与估值折价
- 南向资金与国际资金结构差异带来的估值偏好
- 分红能力和股东回报
- 海外业务、汇率、国际贸易环境
- 控股股东和关联交易治理

## 3. 估值习惯

- 港股更常出现系统性低估，不能只看低 PE 就下结论
- 对高股息公司要把股息稳定性与分红覆盖能力拆开分析
- 对互联网、平台、消费龙头，要分别判断“业务质量”和“港股折价”

## 4. 风险提示

港股特有风险至少考虑：

- 流动性折价
- 国际利率环境变化
- 汇率与外资风险偏好变化
- AH 估值错位持续时间过长
