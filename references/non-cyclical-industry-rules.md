# 非周期性行业规则

适用于消费、品牌零售、互联网平台、软件服务、医疗服务、公用事业等相对不以商品价格周期为主导的行业。

## 1. 核心判断框架

分析非周期行业时，重点不是“周期底部反转”，而是“持续性”和“质量”：

1. 需求是否稳定
2. 定价权是否可持续
3. 护城河是否在强化还是弱化
4. 长期竞争格局是否改善或恶化

## 2. 重点检查项

至少关注：

- 用户 / 客户黏性
- 品牌力或平台效应
- 单位经济模型
- 提价能力与价格传导
- 竞争者进入门槛
- 监管环境变化

## 3. 结论要求

- 不要因为短期波动就误判长期逻辑破坏
- 也不要因为行业“看起来稳定”就忽视增长停滞、护城河退化或监管扰动
- 必须判断公司优势是一次性的，还是可以复利积累的

## 4. 估值约束

- 可优先使用 PE、自由现金流、EV/EBITDA 等常规估值口径
- 需要解释高估值是否由高质量和高持续性支撑
- 对“高质量但高估值”的标的，要把价格与质量拆开判断

## 5. 定价权专项

非周期行业往往更依赖定价权判断：

- 提价是否伤害用户留存
- 提价后毛利率是否改善
- 是否具备品牌、网络效应、转换成本或刚需属性支撑
