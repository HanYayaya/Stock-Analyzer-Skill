# 方法论扩展指南

本文档用于规范 `Stock Analyzer` 新增第二套或多套投资方法论时的接入方式，避免不同方法论之间结构混乱、输出口径不一致，或与现有报告模板冲突。

## 1. 扩展目标

新增方法论时，需要同时满足两件事：

- 保留现有报告的统一章节结构，不因为方法论增加而打散主报告
- 让每一套方法论都能独立给出结论、优缺点、买卖区间和风险判断

换句话说：

- 公共事实层只写一遍
- 方法论结论层分别展开

## 2. 哪些内容是共用的

以下内容属于“公共事实层”，无论启用几套方法论，都只写一次：

- 股票基础信息
- 关键财务与估值数据
- 近 30 天重大事件
- 行业分析
- 公司分析
- 估值分析中的原始事实与口径说明
- 逆向投资与定价权的基础观察
- 数据来源说明
- 交卷前自检摘要

这些内容服务于所有方法论，不要为每一套方法论重复写一遍。

## 3. 哪些内容必须按方法论拆开

以下内容必须在“分方法论投资结论”章节中逐套展开：

- 方法论名称
- 一句话结论
- 核心观点
- 好的部分
- 不好的部分
- 如果买
- 如果不买
- 建议买入价格或区间
- 建议卖出价格或区间
- 倍数到价格的换算逻辑
- 最大风险
- 能力圈检查
- 止损策略
- 止盈策略
- 持有期预期

如果后续启用多套方法论，不得只保留一个合并后的总括结论。

## 4. 新方法论文档建议结构

每新增一种方法论，建议在 `references/` 下新增单独文档，命名格式建议为：

```text
methodology-<slug>.md
```

例如：

```text
methodology-buffett-fcf.md
methodology-peter-lynch-growth.md
methodology-graham-asset-value.md
```

单个方法论文档建议至少包含：

1. 方法论定位
2. 适用市场
3. 适用行业
4. 不适用场景
5. 核心判断问题
6. 关键指标
7. 估值口径偏好
8. 买入触发条件
9. 卖出触发条件
10. 常见误判
11. 与现有方法论的关系

## 5. 新方法论至少要回答的问题

每一套方法论都应明确回答：

- 它更偏重质量、成长、资产还是周期
- 它最看重哪些指标
- 它最容易误判哪些股票
- 它适合在哪些市场使用
- 它适合哪些行业，不适合哪些行业
- 它如何定义“可以买”
- 它如何定义“该卖”

如果这些问题回答不清，这套方法论就不适合接入正式报告。

## 6. 与现有方法论的衔接原则

现有《投资中最简单的事》框架属于：

- 好行业
- 好公司
- 好价格
- 逆向投资
- 定价权

新增方法论时，建议按以下三类方式接入：

### 6.1 补充型

用于补充现有框架的盲区，例如：

- 自由现金流质量
- 资本配置能力
- 成长持续性
- 资产重估价值

这类方法论通常适合作为“第二结论层”。

### 6.2 冲突型

用于给出与现有框架不同的判断，例如：

- 三好原则偏保守，但成长法认为仍可买
- 三好原则认为行业一般，但资产法认为价格极低

这类方法论的价值在于揭示分歧，不在于强行统一结论。

### 6.3 过滤型

用于先筛掉不适合某类方法论处理的股票，例如：

- 周期股不适合直接套用 PEG
- 重资产公司不适合只看 FCF yield
- 金融股不适合照搬制造业 ROIC 口径

过滤型规则应写清楚，避免误用。

## 7. 输出结构要求

新增方法论后，报告中必须同步更新两个位置：

### 7.1 方法论结论总览

在报告最前面新增一行该方法论的一句话结论。

### 7.2 分方法论投资结论

新增该方法论的完整展开内容。

二者缺一不可。

## 8. 评分要求

不是每套方法论都必须强制做百分制评分。

建议分三类处理：

- 强评分型：适合做明确分项打分，例如三好原则
- 弱评分型：可以做等级判断，但不强制精确打分
- 叙述型：重点在逻辑判断，不建议伪精确打分

如果新增方法论没有稳定、可复用的评分规则，不要为了整齐硬做分数。

## 9. 价格建议要求

无论新增哪种方法论，只要给出买入或卖出建议，都必须落到价格：

- 可以直接给价格
- 也可以先给 PE / PB / PS / EV/EBITDA
- 但最终都必须换算成价格或价格区间

同时写清：

- 用了什么口径换算
- 对应的核心参数是什么
- 为什么这个价格区间合理

## 10. 推荐的新增方法论方向

如果要优先新增一套与现有框架互补性强、落地性较高的方法论，建议优先考虑：

- 巴菲特 / 芒格式“护城河 + 自由现金流 + 资本配置”框架
- 彼得林奇式“成长性 + PEG + 业务可理解性”框架
- 格雷厄姆式“资产价值 + 安全边际”框架
- 周期股专用“中枢利润 + PB / 重置成本 + 周期位置”框架

其中：

- 如果你想补“高质量公司”的长期持有判断，优先加巴菲特 / 芒格
- 如果你想补“成长股”的判断，优先加彼得林奇
- 如果你想补“深度价值 / 破净 / 低估资产”，优先加格雷厄姆
- 如果你想强化资源、化工、航运、养殖等，优先加周期专用框架

## 11. 当前建议

在当前版本里，最稳妥的做法是：

1. 先选定第二套方法论
2. 为它新增单独 `reference`
3. 明确它是补充型、冲突型还是过滤型
4. 再把它接入报告总览和“分方法论投资结论”

不要一次并行新增多套方法论，否则很容易在规则口径、估值口径和结论表达上互相打架。
